# 알바이오텍 (RBIOTECH) IR 자료 추출 내용

## 기업 개요
- 기업명: 알바이오텍 주식회사 (RBIOTECH Co., Ltd.)
- 슬로건: "Physical AI for Critical Care"
- 핵심 기술: AI 기반 와상 환자 배변 케어 자동화 — 돌봄 로봇 플랫폼
- 비전: 인간 존엄성을 지키는 배변처리 로봇과 플랫폼 (글로벌 돌봄 분야 리더)
- 주소: 서울특별시 영등포구 경인로 775 에이스하이테크시티
- 홈페이지: www.rbiotech.kr
- 대표전화: 02-2638-0601

## 핵심 문제 정의
1. **간병인/보호자 부담**: 반복된 고강도 혐오 업무(배변처리)로 번아웃, 내국인 신규 진입 불가한 혐오 직업으로 고착
2. **환자 존엄성/프라이버시**: 배변 케어 과정에서 반복적 수치심 경험 → 우울감 증가, 삶의 의지 약화
3. **돌봄 위기 심화**: 돌봄 인구 급증 + 간병 인력 심각한 감소 (24시간 대기, 근골격질환, 욕창, 감염 등)

## 핵심 솔루션: 배변처리 로봇 & 플랫폼
- **Full Cycle Automation System**: 로봇 팔을 통한 기저귀 절개·폴딩·밀봉·신규 삽입 자동화
- **세정 헤드**: 로봇 팔 미세분사(온수), 일회용 스폰지/자동 교체 카트리지, 온풍 건조
- **환자 리프팅**: 비전센싱 → 체위 인식 → HOISTING 경로/각도 산출 → 리프팅 안전한계 체크
- **프라이버시 보호**: 반개방형 차단막으로 시야·노출 최소화 + 국소 음압으로 악취·비말 차단
- **LLM 인터페이스**: LLM 기반 단계별 자연어 대화 안내, 3중 안전 비상정지, 처리 결과 요약·보고
- **자율주행**: 모바일 호출 → 병실 자율도착 → 사용 회수 차감 → 사용로그 → 대시보드
- **KPI 대시보드**: 가동률, 실패율, 평균 세션시간, 불만/민원, 감염지표 모니터링

## 기존 제품 대비 차별점
- 기존 제품(큐라코 등): 고정식 비데형, 중환자 전용, 비인간적 고정자세 요구, 틈새 누수
- 알바이오텍: 인간 중심 접근(평소처럼 침상에서 기저귀만 착용), 인체공학 기술(비전 인식 + 호이스팅), 완벽한 자동화(세정~기저귀 교체까지)

## 장치 처리능력 & 운영모델

## 개발 단계 (4단계 로봇 프로세스)
1. **1단계 (Function PoC)**: 주요 파트 개발 — 로봇 팔 기구 설계·제작, 안전 중심 기본 동작 검증
2. **2단계 (Semi Automation)**: 센서 기반 보조 자동화 — 자동 교체 카트리지, 힘·토크 제한, ROS2 기반
3. **3단계 (Advanced Automation)**: 인지 기반 전자동 세정 — 환자 유형별 전자동 시나리오
4. **4단계 (Full Automation)**: 시제품 상용화 — 의료기기 인허가·임상·양산 전주기
