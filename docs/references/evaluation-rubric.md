# 스타트업 평가 루브릭

> 지원서 스크리닝 및 투자 심사에 사용하는 표준 평가 기준 (100점 만점)

---

## 정량 평가 (60점)

### 1. 팀 역량 (15점)

| 점수 | 기준 |
|------|------|
| 13-15 | 관련 산업 경험 5년 이상, 창업/EXIT 경험, 핵심 역할 전원 풀타임, 기술+사업 보완적 구성 |
| 10-12 | 관련 산업 경험 3년 이상, 핵심 역할 대부분 풀타임, 적절한 보완성 |
| 7-9 | 관련 경험 있으나 제한적, 일부 파트타임, 보완성 부족 영역 존재 |
| 4-6 | 관련 경험 부족, 핵심 역할 공백, 팀 구성 미흡 |
| 1-3 | 경험 매우 부족, 1인 창업, 팀 구성 계획 불명확 |

**세부 평가 항목:**
- 창업팀의 도메인 전문성 (해당 산업에서의 경력 및 인사이트)
- 이전 창업/EXIT 경험 유무
- 핵심 포지션 충원 현황 (CEO, CTO, CPO 등)
- 풀타임 전환 여부 및 시점
- 팀원 간 역량 보완성 (기술 + 비즈니스 + 도메인)
- 공동 창업자 간 관계 및 지분 구조의 합리성

### 2. 시장 규모 및 성장성 (15점)

| 점수 | 기준 |
|------|------|
| 13-15 | TAM 1조원 이상, 연 성장률 20% 이상, SAM/SOM 논리적, 데이터 근거 충분 |
| 10-12 | TAM 5천억 이상, 연 성장률 15% 이상, SAM/SOM 근거 존재 |
| 7-9 | TAM 1천억 이상, 성장 시장이나 근거 부분적 |
| 4-6 | 시장 규모 작거나 정체, 근거 불충분 |
| 1-3 | 시장 규모 불명확, 축소 시장, 근거 없음 |

**세부 평가 항목:**
- TAM(Total Addressable Market) 규모 및 산출 근거
- SAM(Serviceable Available Market) 정의의 합리성
- SOM(Serviceable Obtainable Market) 달성 가능성
- 시장 성장률 및 성장 동인
- 규제 환경 변화에 따른 시장 영향
- 글로벌 확장 가능성

### 3. 트랙션 (15점)

| 점수 | 기준 |
|------|------|
| 13-15 | 유의미한 매출/사용자, MoM 20% 이상 성장, 높은 리텐션, PMF 달성 징후 |
| 10-12 | 초기 매출/사용자 확보, MoM 10% 이상 성장, 리텐션 양호 |
| 7-9 | 베타 사용자 확보, 성장세 존재, 초기 지표 |
| 4-6 | MVP 완성, 소수 파일럿 고객, 성장 미미 |
| 1-3 | 아이디어 단계, 프로토타입 미완성, 트랙션 없음 |

**세부 평가 항목:**
- 월간 활성 사용자(MAU) 또는 월 반복 매출(MRR)
- 월간 성장률(MoM Growth Rate)
- 고객 리텐션율(Retention Rate)
- 유료 전환율(Conversion Rate)
- 핵심 지표의 추세 방향
- PMF(Product-Market Fit) 달성 증거

### 4. 비즈니스 모델 (15점)

| 점수 | 기준 |
|------|------|
| 13-15 | 명확한 수익 모델, 높은 마진, 확장성 입증, 단위 경제학 양호 |
| 10-12 | 수익 모델 정의됨, 마진 합리적, 확장 가능성 있음 |
| 7-9 | 수익 모델 존재하나 검증 부족, 확장성 불확실 |
| 4-6 | 수익 모델 불명확, 마진 낮음, 확장 어려움 |
| 1-3 | 수익 모델 없음, 수익화 계획 부재 |

**세부 평가 항목:**
- 수익 구조의 명확성 (구독, 거래수수료, 광고, 라이선스 등)
- 매출총이익률(Gross Margin)
- 고객획득비용(CAC) 대비 고객생애가치(LTV) 비율
- 투자금 회수 기간(Payback Period)
- 비즈니스 모델의 확장성 (한계비용 체감 여부)
- 가격 결정력 및 경쟁 방어 가능성

## 정성 평가 (40점)

### 5. 문제 정의 명확성 (10점)

| 점수 | 기준 |
|------|------|
| 9-10 | 고객 페인포인트가 명확하고 절실함, 직접 경험 기반, 문제의 크기와 빈도가 큼 |
| 7-8 | 페인포인트 명확, 고객 인터뷰 등 근거 있음 |
| 5-6 | 문제 존재하나 절실도 부족 또는 니치 시장 |
| 3-4 | 문제 정의 모호, "있으면 좋은" 수준 |
| 1-2 | 문제 자체가 불명확하거나 존재하지 않음 |

### 6. 솔루션 차별화 (10점)

| 점수 | 기준 |
|------|------|
| 9-10 | 기술적 해자(moat) 보유, 특허/독점 데이터, 경쟁사 대비 10x 개선 |
| 7-8 | 명확한 차별점, 경쟁사 대비 3x 이상 개선, 모방 장벽 존재 |
| 5-6 | 차별점 존재하나 모방 용이, 경쟁 우위 일시적 |
| 3-4 | 기존 솔루션과 유사, 차별화 미미 |
| 1-2 | 차별점 없음, 레드오션 진입 |

### 7. 실행력 및 속도 (10점)

| 점수 | 기준 |
|------|------|
| 9-10 | 빠른 실행 이력 증명, 피봇 경험, 자원 효율적 활용, 명확한 로드맵 |
| 7-8 | 양호한 실행 속도, 계획 대비 달성률 높음 |
| 5-6 | 보통 수준의 실행력, 일부 지연 |
| 3-4 | 실행 속도 느림, 계획 대비 지연 심각 |
| 1-2 | 실행 이력 없음, 계획만 존재 |

### 8. 액셀러레이터 적합성 (10점)

| 점수 | 기준 |
|------|------|
| 9-10 | 우리 포커스 영역 정합, 기존 포트폴리오와 시너지, 멘토 네트워크 활용 극대화 가능 |
| 7-8 | 포커스 영역 부합, 네트워크 활용 가능 |
| 5-6 | 부분적 부합, 일부 지원 가능 |
| 3-4 | 포커스 영역 불일치, 지원 효과 제한적 |
| 1-2 | 완전 불일치, 지원 가치 없음 |

## 종합 판정 기준

| 총점 | 판정 | 권고 액션 |
|------|------|----------|
| 80-100 | **통과 (Pass)** | 2차 심사(미팅) 진행 권고 |
| 70-79 | **조건부 통과 (Conditional)** | 특정 조건 확인 후 진행 |
| 50-69 | **보류 (Hold)** | 추가 자료 요청 또는 다음 배치 검토 |
| 0-49 | **탈락 (Decline)** | 사유를 명시하여 정중히 안내 |

## 추가 체크 항목 (Pass/Fail)

- [ ] 법적 결격사유 없음
- [ ] 동일/유사 서비스 중복 지원 아님
- [ ] 대표자 신용 이슈 없음
- [ ] 기존 포트폴리오사와 직접 경쟁 관계 아님
- [ ] 반사회적 사업 모델 아님

---

## 평가자 유의사항

1. **편향 주의**: 특정 산업/배경에 대한 선입견 배제
2. **근거 기반**: 모든 점수에 구체적 근거를 기술
3. **불확실성 명시**: 확인 불가한 사항은 "[미확인]"으로 표기
4. **상대 평가 지양**: 각 기업을 절대 기준으로 평가
5. **시점 고려**: 초기 스타트업의 특성(데이터 부족 등)을 감안
