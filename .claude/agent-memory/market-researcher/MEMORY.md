# market-researcher 에이전트 메모리

## 조사 방법론

### 시장 조사 핵심 원칙
- 특허 정보 서비스 시장은 세분화(검색/분석/관리)에 따라 시장 규모 편차가 매우 큼
- 국내 시장 규모는 직접 데이터가 드물어 글로벌 비중(특허 출원 기준 약 6.7%) 역산이 유효
- NPE 소송 트렌드는 특허 분석 수요와 강한 상관관계를 가짐 (2024-2025년 증가세 확인)

### WebSearch 효과적 쿼리 패턴
- "{산업} market size {연도} CAGR" (영문이 국문보다 더 많은 수치 데이터 반환)
- "{경쟁사명} revenue 2024 2025" (구체적 재무 데이터 검색 시 영문 우선)
- "WIPO {연도} patent filing statistics" (공신력 있는 출원 통계)
- 국내 시장은 한글 쿼리로 별도 검색 필요하나 수치 데이터 희소

### TAM 검증 공통 주의사항
- IR의 TAM이 완제품/상위 시장 기준일 때 실제 B2B 모듈/서비스 레이어 TAM은 0.1~0.5% 수준
- 전체 시장을 TAM으로 주장하면 실제 SAM과 괴리 — Bottom-up 검증 필수
- 시장 정의 범위(하드웨어 포함, 소프트웨어만, 서비스 포함)에 따라 3~10배 편차
- 같은 시장도 리서치 펌마다 수치 상이 → 복수 출처 교차 확인 권장

### 데이터 신뢰도 등급
- WIPO 공식 통계: 최고 신뢰도
- Clarivate/LexisNexis 공식 IR: 높음
- SkyQuestt, Fortune Business Insights 등 리서치 펌 수치: 출처별 편차 있으므로 복수 교차 확인 권장
- getlatka.com 수익 데이터: 참고용 (자체 추정치 포함 가능)
