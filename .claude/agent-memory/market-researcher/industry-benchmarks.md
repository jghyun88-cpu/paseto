### 역사 콘텐츠 IP 시장 조사 시 주의사항

- DB 독점성 검증이 핵심: 공공 자료 재가공 vs 진정한 독점 자료 구분 필요
- 동명 기업 혼동 주의: 광개토연구소(Patentpia, 2002년 설립, 강민수 대표)는 특허DB 기업으로 역사콘텐츠와 무관

### 탄소섬유 발열체 시장 조사 시 주의사항
- 탄소섬유 히팅케이블 시장은 피복 소재(PTFE/실리콘/PVC)에 따라 별도 집계되어 수치 혼재 → 반드시 세그먼트 정의 확인
- 전기 히팅케이블 "전체 시장"을 TAM으로 주장하는 경우 실제 도달 가능 시장(SAM)과 괴리 큼 (산업용 배관 추적 포함 여부 등)
- 2015년 시점 탄소섬유 히팅케이블 독립 시장 데이터는 공개 자료 극히 희소 → 추정치 사용 불가피하며 명시 필수
- 한국 농업용 히팅케이블 시장은 별도 통계 없음 → 시설원예 면적 × 전기 난방 전환율 × 단가 Bottom-up 검증 필요
- 자동차 시트열선 OEM 진입 심사 시: IATF 16949 인증 여부, Tier-1 공급망 편입 현황 필수 확인

### 로봇/WPT 시장 조사 시 주의사항
- WPT TAM 제시 시 소비자(Qi)·산업·EV 세그먼트 명확 분리 필수 → 전체 WPT TAM을 로봇 무선충전 TAM으로 혼용하면 100배 이상 과대
- BCG 로봇 시장 $2,600억은 전체 로봇 생태계(SW·서비스 포함) 낙관 시나리오 상단값. 하드웨어 무선충전 레이어는 이의 0.1~0.5% 수준
- 85kHz 대역은 표준 ISM 대역이 아니어서 해외 국가별 전파 인증 복잡. CE/FCC 진출 시 추가 검토 필수
- 국내 로봇 무선충전 시장은 2025.03 규제 개정 이후 급성장 초입으로 공개 데이터 희소 → Bottom-up 추정 필수
