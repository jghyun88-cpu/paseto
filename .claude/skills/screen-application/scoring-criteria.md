# 스크리닝 스코어링 기준 요약

> application-screener 에이전트의 빠른 참조용 스코어링 가이드

## 점수 배점 구조

```
총 100점 = 정량 평가 60점 + 정성 평가 40점

[정량 평가 60점]
├── 팀 역량         15점
├── 시장 규모/성장성  15점
├── 트랙션          15점
└── 비즈니스 모델    15점

[정성 평가 40점]
├── 문제 정의 명확성  10점
├── 솔루션 차별화     10점
├── 실행력/속도      10점
└── AC 적합성       10점
```

## 빠른 판정 기준

| 총점 | 판정 | 다음 단계 |
|------|------|----------|
| 80+ | 통과 | 2차 심사(미팅) 진행 |
| 70-79 | 조건부 | 조건 확인 후 진행 |
| 50-69 | 보류 | 추가 자료 요청 |
| 0-49 | 탈락 | 사유 안내 |

## 즉시 탈락 사유 (Pass/Fail)

- 법적 결격사유
- 반사회적 사업 모델
- 포트폴리오 직접 경쟁
- 사기/허위 정보

## 산업별 가중치 참고

| 산업 | 가중치 높은 항목 |
|------|----------------|
| 딥테크/AI | 팀 기술역량, 솔루션 차별화 |
| B2C 플랫폼 | 트랙션 (MAU, 리텐션) |
| B2B SaaS | 비즈니스 모델 (단위 경제학) |
| 바이오/헬스 | 팀 전문성, 규제 대응 |
| 핀테크 | 비즈니스 모델, 규제 준수 |
