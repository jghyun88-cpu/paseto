# 투자 검토 메모 작성 프로세스

대상 기업: $ARGUMENTS

## 실행 절차

### 1. 기존 분석 자료 확인

1. `ai-agent/reports/screening/` 에서 대상 기업의 스크리닝 결과를 검색합니다
2. `ai-agent/reports/analysis/` 에서 대상 기업의 IR 분석 보고서를 검색합니다
3. `ai-agent/data/applications/` 에서 지원서 원본을 확인합니다
4. 기존 자료가 충분한지 판단합니다

### 2. 추가 분석 실행 (기존 자료 부족 시)

기존 IR 분석 보고서가 없거나 30일 이상 경과된 경우, 3개 에이전트를 **병렬 실행**합니다:

**병렬 에이전트:**
- `ir-analyst`: IR 자료 구조 및 내용 심층 분석
- `market-researcher`: 시장 규모 검증, 경쟁 환경 최신화
- `financial-analyst`: 재무 추정 검증, 밸류에이션 분석

### 3. 투자 메모 구성

`.claude/skills/investment-memo/memo-template.md` 를 참조하여 다음 섹션을 작성합니다:

#### 3.1 투자 요약 테이블
- 기업명, 라운드, 투자 금액, 밸류에이션, 목표 지분율, 투자 형태

#### 3.2 Investment Thesis (투자 근거)
- 왜 이 기업에 투자해야 하는가를 3-5문장으로 명확히 작성
- 시장 기회 + 팀 역량 + 트랙션 + 타이밍을 연결

#### 3.3 기업 분석 요약
- 기존 스크리닝/IR 분석 결과를 투심위 관점으로 재구성
- 비즈니스 모델, 시장 기회, 트랙션, 팀 역량 각 1-2 문단

#### 3.4 리스크 및 완화 방안
- 핵심 리스크를 영향도/발생확률 매트릭스로 분류
- 각 리스크에 대한 구체적 완화 방안 제시

#### 3.5 투자 조건 상세
- 주요 투자 조건 정리
- 특수 조건 (우선참여권, 반희석화, 이사회 구성 등)
- 마일스톤 기반 조건 (해당 시)

#### 3.6 기대 수익 시나리오
- 낙관/기본/비관 3가지 시나리오
- 각 시나리오별 가정, EXIT 밸류, MOIC 추정
- `.claude/rules/investment-criteria.md` 의 밸류에이션 가이드라인 참조

#### 3.7 유사 투자 사례
- 동일 섹터/스테이지의 과거 투자 사례 참조
- 벤치마크 밸류에이션 비교

#### 3.8 투심위 논의 포인트
- 투심위에서 반드시 논의해야 할 핵심 쟁점 5-7개
- 각 쟁점에 대한 찬/반 논점 정리

### 4. 검토 및 저장

1. `.claude/rules/reporting-standards.md` 형식 준수 확인
2. `.claude/rules/investment-criteria.md` 투자 기준 부합 여부 체크
3. 메타 정보 블록 작성 (기밀등급: 대외비)
4. `ai-agent/reports/investment-committee/{기업명}_memo_{날짜}.md` 에 저장

### 5. 결과 요약 보고

사용자에게 다음을 보고합니다:
- Investment Thesis 핵심 요약
- 투자 매력도 (5점 척도)
- 핵심 리스크 Top 3
- 기대 수익 (기본 시나리오 MOIC)
- 투심위 핵심 논의 포인트
- 저장 경로

## 주의사항

- 투자 메모는 **대외비** 등급입니다
- 긍정적 측면과 리스크를 **균형** 있게 서술합니다
- 추정/가정은 명확히 표기하고 근거를 제시합니다
- 기존 분석 보고서의 인사이트를 최대한 활용합니다
- 최종 투자 판단은 투심위에서 결정됨을 명시합니다
