# 반도체와 모빌리티를 한 장에서 같이 볼 때의 읽는 기준

이 포맷은 단순 현황표가 아니라 **산업형 의사결정표**입니다.
대표는 아래 순서로 보면 됩니다.

## 1. TRL

기술이 제자리인지, 실제 검증 단계로 올라가는지 확인

## 2. PoC

기술이 고객사 검증 단계로 넘어가는지 확인

## 3. TIPS

민간투자를 정부 R&D로 확대할 수 있는지 확인

## 4. CVC

전략적 자본과 고객을 동시에 붙일 수 있는지 확인

즉 이 보고서의 핵심 질문은 이것입니다.

**“이 기업이 기술검증 → 실증 → 정책연계 → 전략투자까지 이어질 가능성이 얼마나 높아졌는가?”**

# 반도체용과 모빌리티용을 볼 때의 차이

## 반도체에서 대표가 특히 봐야 할 것

* TRL 상승보다 **고객 공정 적용성**
* PoC보다 **샘플테스트와 수율 영향**
* CVC보다 **공동개발/라인 검증 가능성**
* TIPS보다 **장기 R&D 마일스톤 정합성**

## 모빌리티에서 대표가 특히 봐야 할 것

* 기술 성능보다 **실차/현장 적용성**
* PoC보다 **안전성·연동성**
* CVC보다 **OEM/Tier 구조 적합성**
* TIPS보다 **규제·인증 포함 사업화 계획**

그래서 같은 보고서라도 대표 코멘트는 산업별로 다르게 적는 것이 좋습니다.
