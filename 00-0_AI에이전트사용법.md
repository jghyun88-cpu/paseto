  1. 스크리닝 보고서 작업

  Claude Code에서 다음과 같이 요청합니다:

  "배터와이 지원서를 스크리닝해줘"
  또는
  "/screen-application 배터와이"

  AI가 수행하는 단계:

  1. 데이터 수집 — MCP 도구로 eLSA DB에서 기업 정보를 가져옴
    - elsa_get_startup → 기업 기본정보
    - elsa_list_screenings → 기존 스크리닝 결과 확인
  2. 배치 설정 로드 — ai-agent/data/batch-config/batch-2026-H1.md에서 가중치 확인
    - 팀 역량: 20점, 시장 규모: 15점, 제품/기술: 18점, 비즈니스 모델: 7점
  3. 평가 수행 (100점 만점)

  ┌─────────────┬─────────────────────┬──────┐
  │    구분     │        항목         │ 배점 │
  ├─────────────┼─────────────────────┼──────┤
  │ 정량 (60점) │ 팀 역량             │ 20점 │
  ├─────────────┼─────────────────────┼──────┤
  │             │ 시장 규모 및 성장성 │ 15점 │
  ├─────────────┼─────────────────────┼──────┤
  │             │ 제품/기술           │ 18점 │
  ├─────────────┼─────────────────────┼──────┤
  │             │ 비즈니스 모델       │ 7점  │
  ├─────────────┼─────────────────────┼──────┤
  │ 정성 (40점) │ 비전 및 확장성      │ 15점 │
  ├─────────────┼─────────────────────┼──────┤
  │             │ 솔루션 차별화       │ 12점 │
  ├─────────────┼─────────────────────┼──────┤
  │             │ 투자 적합성         │ 8점  │
  ├─────────────┼─────────────────────┼──────┤
  │             │ 프리젠테이션        │ 5점  │
  └─────────────┴─────────────────────┴──────┘

  4. 판정 기준

  ┌────────┬────────────────┬────────────────────────┐
  │  총점  │      판정      │          액션          │
  ├────────┼────────────────┼────────────────────────┤
  │ 80~100 │ 통과 (Pass)    │ 2차 심사 진행          │
  ├────────┼────────────────┼────────────────────────┤

  6. DB 저장 및 조회

  모든 분석 결과는 MCP 도구를 통해 DB에 자동 저장됩니다:

  elsa_submit_ai_analysis 호출:
  {
    startup_id: "기업 UUID",
    analysis_type: "screening | ir_analysis | market_scan | ...",
    scores: { "team": 14, "market": 13, ... },
    summary: "74점, 조건부 통과...",
    report_path: "reports/screening/배터와이_screening_2026-03-21.md",
    risk_level: "low | medium | high | critical",
    recommendation: "pass | conditional | hold | decline",
    investment_attractiveness: 3  (1~5점)
  }

  저장된 결과는 백엔드 API로 조회 가능:
  - GET /api/v1/ai-analysis/?startup_id={id} — 기업별 전체 분석 결과
  - GET /api/v1/ai-analysis/{id} — 단건 조회
